# Air Canvas Pro

A real-time **virtual drawing application** built with **Python, OpenCV, MediaPipe, and NumPy**.

This project uses hand tracking to let you draw in the air with your finger, select colors from a toolbar, erase strokes, undo actions, save drawings, and adjust brush size using pinch gestures.

## Features

- Real-time hand tracking with MediaPipe
- Air drawing using index finger
- Toolbar for color selection
- Eraser mode
- Clear canvas
- Undo support
- Save drawing as image
- Pinch gesture for brush size control
- Smooth drawing motion
- FPS display
- Webcam-based interaction

## Tech Stack

- **Python**
- **OpenCV**
- **MediaPipe**
- **NumPy**

## Controls

### Hand Gestures
- **Selection Mode:** Index + Middle finger up
- **Drawing Mode:** Index finger up only
- **Pinch Gesture:** Adjust brush size while drawing

### Keyboard Shortcuts
- **Q** → Quit
- **C** → Clear canvas
- **U** → Undo
- **S** → Save drawing
